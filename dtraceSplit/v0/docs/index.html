dtrsplit

Used to split Daikon trace files (.dtrace) into a separate declaration files and the remaining data files.

Usage

dtrsplit [file]+
This split the given Daikon trace files (.dtrace) into a separate declaration files and the remaining data files. For each f.dtrace input file, its program points declration part is separated into a new file f.decls. The file f.dtrace itself will now contain only the log data of the original program points.
