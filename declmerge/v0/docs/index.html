declmerge

Used to merge several Daikon declaration files (.decls) into a single .decl file.

License: BSD3
Authors: Wishnu Prasetya. The development is part of the EU-funded FITTEST project.
Copyright (C) Utrecht University, 2010.

Usage

declmerge [file]+
This will merge the given .decls files into one merged.decls file. A program point (ppt) that occurs in one of the .decl will occur in the merged file, and will occur only once.
